Basic Operators in C

Definition: An operator is a symbol that tells the compiler to perform specific mathematical, logical, or relational operations on operands (values or variables). Operators are the building blocks for creating expressions and performing calculations in C.
Key Terms:
  • Operand: The values or variables on which an operation is performed
  • Expression: A combination of operators and operands that evaluates to a value
  • Precedence: The order in which operations are performed

Let's start with the simplest operations in C programming. We'll begin with very basic examples and gradually work our way up to more complex ones.

SUPER EASY START: Your First Calculation

Before we learn about different types of operators, let's start with the most basic math:

Program Definition: This program demonstrates arithmetic operations in C by performing a simple addition and displaying the result. It shows how C can evaluate mathematical expressions directly within the printf() function.
Key Programming Concepts:
  • Arithmetic Expression: 2 + 3 is evaluated at compile time and replaced with the result
  • Format Specifier: %d tells printf() to expect an integer value
  • Direct Calculation: The math is performed automatically by the compiler
  • Immediate Output: Result is calculated and displayed in one statement
#include <stdio.h>

int main() {
    printf("2 + 3 = %d\n", 2 + 3);
    return 0;
}

Output:

2 + 3 = 5

Definition: Input refers to data entered into a program (usually from keyboard), while output refers to data displayed by the program (usually to screen). In C, we use scanf() for input and printf() for output.

Basic Decision Making

Definition: Decision making in programming refers to the ability of a program to choose different paths of execution based on certain conditions. This is implemented using conditional statements that evaluate expressions and execute different blocks of code accordingly.
Key Concepts:
  • Condition: An expression that evaluates to true (1) or false (0)
  • Boolean Logic: True/False reasoning used in conditions
  • Control Flow: The order in which program instructions are executed

Control Flow

Control flow statements allow your program to execute certain blocks of code only if specific conditions are met. This is how programs make decisions.

Structure When to Use Syntax Best For
if Single condition if (condition) { } Simple yes/no decisions
if-else Two alternatives if (condition) { } else { } Either/or situations
if-else if-else Multiple conditions if (c1) { } else if (c2) { } else { } Multiple exclusive choices

1. The if Statement

The if statement runs a block of code only when its condition is true (evaluates to 1).

Syntax:

if (condition) {
    // code to run if condition is true
}

Example 25: Check if Number is Positive

#include <stdio.h>

int main() {
    int number;
    printf("Enter a number: ");
    scanf("%d", &number);

    if (number > 0) {
        printf("The number is positive.\n");
    }

    printf("End of program.\n");
    return 0;
}
Enter a number: 15 The number is positive. End of program.

2. The if-else Statement

The if-else statement provides an alternative block of code to run if the condition is false.

Syntax:

if (condition) {
    // code to run if condition is true
} else {
    // code to run if condition is false
}

3. The if-else if-else Ladder

This structure lets you check multiple conditions in sequence.

Syntax:

if (condition1) {
    // code for condition1
} else if (condition2) {
    // code for condition2
} else {
    // code if no conditions are met
}

Example 39: Leap Year Checker

Write a program to check if a given year is a leap year. A year is a leap year if:

  • It's divisible by 4 AND
  • If it's divisible by 100, then it must also be divisible by 400

Solution:

#include <stdio.h>

int main() {
    int year;
    
    printf("Enter a year: ");
    scanf("%d", &year);
    
    if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)) {
        printf("%d is a leap year.\n", year);
    } else {
        printf("%d is not a leap year.\n", year);
    }
    
    return 0;
}
Enter a year: 2024 2024 is a leap year.
