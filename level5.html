Common Mistakes and Debugging Tips

1. Common Syntax Errors

  • Forgetting semicolons: Every statement in C must end with ;
  • Mismatched braces: Every { must have a corresponding }
  • Case sensitivity: printf is not the same as Printf
  • Missing quotes: Strings must be in double quotes "text", characters in single quotes 'c'

2. Logic Errors

  • Off-by-one errors in loops: Remember arrays start at index 0
  • Infinite loops: Always ensure loop conditions will eventually become false
  • Wrong operator: Using = (assignment) instead of == (comparison)

3. Memory and Array Issues

  • Array bounds: Don't access array[5] if array size is 5 (valid indices: 0-4)
  • Uninitialized variables: Always initialize variables before using them
  • Buffer overflow: Be careful with string operations to avoid writing beyond array bounds

4. Debugging Strategies

  • Use printf statements: Add temporary print statements to see variable values
  • Test small parts: Break your program into smaller functions and test each
  • Read error messages carefully: Compiler errors often tell you exactly what's wrong
  • Use consistent indentation: Makes it easier to spot logic errors

Example of Common Mistakes:

WRONG - Common mistakes

// WRONG - Common mistakes
#include <stdio.h>

int main() {
    int x  // Missing semicolon
    x = 5
    
    if (x = 5) {  // Should be == for comparison
        printf("x is 5\n")  // Missing semicolon
    }
    
    int arr[5];
    arr[5] = 10;  // Out of bounds! Valid indices: 0-4
    
    return 0
}

CORRECT - Fixed version

// CORRECT - Fixed version
#include <stdio.h>

int main() {
    int x;  // Added semicolon
    x = 5;  // Added semicolon
    
    if (x == 5) {  // Correct comparison operator
        printf("x is 5\n");  // Added semicolon
    }
    
    int arr[5];
    arr[4] = 10;  // Correct index (0-4 are valid)
    
    return 0;  // Added semicolon
}

1. Basic Conditional Programs

Example 63: Write a program to check whether a person is eligible to vote or not.

Program Definition: This program demonstrates conditional logic using the if-else statement in C. It takes user input, evaluates a condition, and provides different outputs based on the evaluation result. This showcases decision-making in programming.
Key Conditional Programming Concepts:
  • Input Validation: Using scanf() to receive user input for processing
  • Conditional Logic: if-else statement evaluates a boolean condition (age >= 18)
  • Comparison Operator: >= compares two values and returns true (1) or false (0)
  • Program Flow Control: Execution follows different paths based on condition evaluation
  • Real-world Application: Demonstrates how programming logic applies to practical scenarios
#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);

    if (age >= 18) {
        printf("You are eligible to vote.\n");
    } else {
        printf("You are not eligible to vote.\n");
    }

    return 0;
}
