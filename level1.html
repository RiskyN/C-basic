Introduction to C Programming

Definition: C is a general-purpose, procedural programming language developed by Dennis Ritchie at Bell Labs between 1969 and 1973. It is widely used for system programming, embedded systems, and application development.

What is Programming? Programming is the process of creating a set of instructions that tell a computer how to perform a task.

Key Features of C

  • Fast performance – C programs run quickly because they're turned into instructions the computer directly understands.
  • Organized code – You write code in chunks called functions, which makes it easier to manage.
  • Direct memory access – C lets you work closely with the computer's memory using pointers (a powerful but tricky concept).
  • Works everywhere – C programs can run on many types of computers, from laptops to tiny devices.

Where is C Used?

  • Building operating systems (like UNIX)
  • Writing software for small devices (embedded systems)
  • Making tools like compilers and interpreters

Example 1: Your First Program

EASY Practice Task: Print a message to the screen.
Program Definition: This is a basic C program that demonstrates the fundamental structure of a C program. It uses the printf() function to display text output to the console screen.
Key Components Explained:
  • #include <stdio.h> - Preprocessor directive that includes the standard input/output library
  • int main() - Main function where program execution begins, returns an integer value
  • printf() - Library function that prints formatted text to the screen
  • \n - Escape sequence that creates a new line
  • return 0; - Indicates successful program termination to the operating system
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

When you run this program, it shows:

Hello, World!

Congratulations! You just wrote your first C program!

Steps to Run

  1. Type the code in C-Free
  2. Click Compile (or press F5)
  3. Click Run (or press Ctrl+F5)

Variables and Format Specifiers

Definition: Variables store data that can be modified during program execution. Format specifiers are placeholders that specify the type of data in functions like printf() and scanf().

1. Introduction

C lets you store and display information using variables. Each variable has a data type that tells the computer what kind of value it holds.

2. Integers (int) and %d

The int type stores whole numbers.

#include <stdio.h>

int main() {
    int age = 21;
    printf("Age: %d\n", age);
    return 0;
}

Output:

Age: 21

3. Floating Point Numbers (float) and %f

The float type stores decimal numbers.

#include <stdio.h>

int main() {
    float price = 55.75;
    printf("Price: %f\n", price);
    return 0;
}

Output:

Price: 55.750000

Controlling Decimal Places (%.nf)

You can show only a certain number of decimal places.

#include <stdio.h>

int main() {
    float mark = 92.456;
    printf("Mark: %.2f\n", mark);
    return 0;
}

Output:

Mark: 92.46

4. Characters (char) and %c

The char type stores a single letter or symbol.

#include <stdio.h>

int main() {
    char grade = 'A';
    printf("Grade: %c\n", grade);
    return 0;
}

Output:

Grade: A

5. Double (double) and %f / %.nf

The double type stores large or more precise decimal numbers. You print doubles like floats, using %f and %.nf.

#include <stdio.h>

int main() {
    double weight = 56.78324;
    printf("Weight: %.3f\n", weight);
    return 0;
}

Output:

Weight: 56.783

6. Strings and %s

Strings are arrays of characters (introduced fully later). Still, you can print them using %s.

#include <stdio.h>

int main() {
    char name[] = "Sam";
    printf("Name: %s\n", name);
    return 0;
}

Output:

Name: Sam

Introduction to scanf() in C

scanf() reads input from the keyboard. It requires format specifiers like %d, %f, %c, %s matching the type of variable you want to fill. Use the address-of operator & to provide the variable's memory location.

Example 4: Reading an Integer

#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("Your age is: %d\n", age);
    return 0;
}

Sample Output:

Enter your age: 25 Your age is: 25

Example 5: Reading Multiple Inputs

Here's an easier example for beginners that uses scanf to read two integers and one float, with clear prompts and simple printing:

Example: Reading Two Integers and One Float

#include <stdio.h>

int main() {
    int age;
    int marks;
    float height;

    printf("Enter your age: ");
    scanf("%d", &age);

    printf("Enter your marks: ");
    scanf("%d", &marks);

    printf("Enter your height (in meters): ");
    scanf("%f", &height);

    printf("\nYou entered:\n");
    printf("Age: %d\n", age);
    printf("Marks: %d\n", marks);
    printf("Height: %.2f meters\n", height);

    return 0;
}

Sample Output:

Enter your age: 22 Enter your marks: 95 Enter your height (in meters): 1.75 You entered: Age: 22 Marks: 95 Height: 1.75 meters

Story using basic C data types:

Example 6: Story without user input

#include <stdio.h>

Example 9: Story with user input

A simple story with user input using scanf for all variables:

#include <stdio.h>

int main() {
    char name[50];
    char favoriteFruit[20];
    int age;
    float height;

    printf("Enter a name: ");
    scanf("%s", name);

    printf("Enter favorite fruit: ");
    scanf("%s", favoriteFruit);

    printf("Enter age: ");
    scanf("%d", &age);

    printf("Enter height (in meters): ");
    scanf("%f", &height);

    printf("\nThere was a girl named %s.\n", name);
    printf("She liked to eat %s.\n", favoriteFruit);
    printf("She was %d years old and %.1f meters tall.\n", age, height);

    return 0;
}

Sample Output:

Enter a name: Alice Enter favorite fruit: apple Enter age: 12 Enter height (in meters): 1.4 There was a girl named Alice. She liked to eat apple. She was 12 years old and 1.4 meters tall.
