Definition: Control structures are programming constructs that control the flow of execution in a program. They determine which statements are executed, how many times, and in what order.
Types of Control Structures:
  • Sequential: Statements executed one after another (default)
  • Selection: Choosing between different paths (if, switch)
  • Iteration: Repeating statements multiple times (loops)

Introduction to Loops

Definition: A loop is a control structure that repeatedly executes a block of statements as long as a specified condition remains true. Loops allow you to avoid writing repetitive code.
Key Terms:
  • Iteration: One complete execution of the loop body
  • Loop Counter: A variable that tracks how many times the loop has executed
  • Infinite Loop: A loop that never terminates (usually a programming error)

SUPER EASY: What Are Loops?

Sometimes we want to repeat the same thing many times. Instead of writing the same code over and over, we use loops!

Without Loop (Bad Way):

#include <stdio.h>

int main() {
    printf("Hello!\n");
    printf("Hello!\n");
    printf("Hello!\n");
    printf("Hello!\n");
    printf("Hello!\n");
    return 0;
}

With Loop (Good Way):

Program Definition: This program demonstrates the power of loops in C programming. It uses a for loop to repeat the same action multiple times, eliminating code duplication and making the program more efficient and maintainable.
Key Loop Components Explained:
  • Loop Initialization: i = 1 sets the starting value for the counter variable
  • Loop Condition: i <= 5 determines when the loop should continue executing
  • Loop Increment: i++ increases the counter by 1 after each iteration
  • Loop Body: The code inside braces {} that gets repeated
  • Iteration: Each time the loop executes is called one iteration
#include <stdio.h>

int main() {
    int i;
    for (i = 1; i <= 5; i++) {
        printf("Hello!\n");
    }
    return 0;
}

Both programs do the same thing, but the second one is much better!

LEVEL 3B: Understanding the Loop Parts

Let's break down the for loop:

for (i = 1; i <= 5; i++) {
    // code to repeat
}
  • i = 1 → Start with i = 1
  • i <= 5 → Keep going while i is 5 or less
  • i++ → Add 1 to i each time
  • The code inside { } runs each time

Advanced Control Flow

Definition: Advanced control flow refers to more complex decision-making structures that handle multiple conditions, nested decisions, and sophisticated program logic beyond simple if-else statements.
Advanced Concepts:
  • Nested Conditions: if statements inside other if statements
  • Multiple Conditions: Using logical operators to combine conditions
  • Switch Statements: Efficient way to handle multiple specific values

Now that you understand basic decisions and loops, let's learn more advanced techniques:

4. The switch Statement

The switch statement is used when you need to compare a variable against many different values. It's often cleaner than using multiple if-else if statements.

Syntax:

switch (variable) {
    case value1:
        // code to run if variable equals value1
        break;
    case value2:
        // code to run if variable equals value2
        break;
    default:
        // code to run if no cases match
        break;
}

Example 55: Simple Calculator Using Switch
#include <stdio.h>

int main() {
    float num1, num2, result;
    char operator;
    
    printf("Enter first number: ");
    scanf("%f", &num1);
    printf("Enter operator (+, -, *, /): ");
    scanf(" %c", &operator);
    printf("Enter second number: ");
    scanf("%f", &num2);
    
    switch (operator) {
        case '+':
            result = num1 + num2;
            printf("%.2f + %.2f = %.2f\n", num1, num2, result);
            break;
        case '-':
            result = num1 - num2;
            printf("%.2f - %.2f = %.2f\n", num1, num2, result);
            break;
        case '*':
            result = num1 * num2;
            printf("%.2f * %.2f = %.2f\n", num1, num2, result);
            break;
        case '/':
            if (num2 != 0) {
                result = num1 / num2;
                printf("%.2f / %.2f = %.2f\n", num1, num2, result);
            } else {
                printf("Error: Division by zero!\n");
            }
            break;
        default:
            printf("Invalid operator!\n");
            break;
    }
    
    return 0;
}
Example 56: Vowel or Consonant Using Switch
#include <stdio.h>

int main() {
    char ch;
    
    printf("Enter a character: ");
    scanf(" %c", &ch);
    
    switch (ch) {
        case 'a':
        case 'e':
        case 'i':
        case 'o':
        case 'u':
        case 'A':
        case 'E':
        case 'I':
        case 'O':
        case 'U':
            printf("%c is a vowel.\n", ch);
            break;
        default:
            if ((ch >= 'a' && ch <= 'z') || (ch >= 'A' && ch <= 'Z')) {
                printf("%c is a consonant.\n", ch);
            } else {
                printf("%c is not a letter.\n", ch);
            }
            break;
    }
    
    return 0;
}
