Functions: Reusable Code Blocks

Definition: A function is a self-contained block of code that performs a specific task. It can accept input parameters, process them, and optionally return a result. Functions promote code reusability and modularity.
Key Terms:
  • Function Declaration: Tells the compiler about function name, return type, and parameters
  • Function Definition: Contains the actual body/implementation of the function
  • Function Call: Executing/invoking a function from another part of the program
  • Parameters: Input values passed to a function
  • Return Value: Output value sent back by a function

A function is a block of code that performs a specific task. You've already been using one: main()! Creating your own functions helps you organize your code and reuse it without copying and pasting.

Why use functions?

  • Organization: Break down a large program into smaller, manageable pieces.
  • Reusability: Write a piece of code once and use it many times.
  • Readability: Well-named functions make your code easier to understand.

1. Defining and Calling a Simple Function

A function that doesn't take any input or return any output is the simplest kind. The keyword void is used to indicate this.

Syntax:

void functionName() {
    // code to execute
}

Example 23: Simple Function Example

Program Definition: This program demonstrates user-defined functions in C. It shows how to create a custom function (sayHello) separate from main() and how to call it. This illustrates the concept of modular programming where code is organized into reusable blocks.
Key Function Concepts Explained:
  • Function Definition: void sayHello() creates a new function that returns nothing
  • Function Call: sayHello(); executes the function from within main()
  • Program Flow: Execution starts in main(), jumps to sayHello() when called, then returns to main()
  • Modular Design: Separating functionality into different functions improves code organization
  • Code Reusability: The same function can be called multiple times from different places
#include <stdio.h>

// 1. Define the function
void sayHello() {
    printf("Hello from a function!\n");
}

int main() {
    printf("Starting the main program.\n");
    sayHello(); // 2. Call the function
    printf("Finished the main program.\n");
    return 0;
}

Output:

Starting the main program. Hello from a function! Finished the main program.

2. Functions with Parameters and Return Values

Functions can accept inputs (called parameters) and produce an output (called a return value). This makes them much more powerful.

Syntax:

returnType functionName(dataType parameter1, dataType parameter2) {
    // code that uses parameters
    return value; // value must match the returnType
}

Example 57: A function that adds two numbers

#include <stdio.h>

// This function takes two integers as input and returns their sum (an integer)
int add(int num1, int num2) {
    int sum = num1 + num2;
    return sum;
}

int main() {
    int result = add(10, 5); // Call the function and store its return value
    printf("The result of the addition is: %d\n", result);

    int anotherResult = add(100, 200);
    printf("Another result is: %d\n", anotherResult);
    
    return 0;

The result of the addition is: 15 Another result is: 300

Example 58: Area of a Rectangle

Write a C program that includes a function named calculateArea. This function should:

  • Take two integer parameters: length and width.
  • Calculate the area (length * width).
  • Return the calculated area as an integer.

In your main function, call calculateArea with some values (e.g., length 8 and width 5) and print the returned result.

Solution to Example 58:

#include <stdio.h>

int calculateArea(int length, int width) {
    int area = length * width;
    return area;
}

int main() {
    int rectLength = 8;
    int rectWidth = 5;

    // Call the function and store the result
    int areaResult = calculateArea(rectLength, rectWidth);

    return 0;
}
The area of a rectangle with length 8 and width 5 is 40.

Example 60: Prime Number Checker

Write a function isPrime(int num) that returns 1 if the number is prime, 0 otherwise.

Solution:
#include <stdio.h>

int isPrime(int num) {
    if (num <= 1) {
        return 0;  // Not prime
    }
    
    // Check if num is divisible by any number from 2 to num-1
    for (int i = 2; i < num; i++) {
        if (num % i == 0) {
            return 0;  // Not prime
        }
    }
    return 1;  // Prime
}

int main() {
    int number;
    
    printf("Enter a number: ");
    scanf("%d", &number);
    
    if (isPrime(number)) {
        printf("%d is a prime number.\n", number);
    } else {
        printf("%d is not a prime number.\n", number);
    }
    
    return 0;
}

Example 61: Factorial Function

Write a function that calculates factorial using recursion.

Solution:
#include <stdio.h>

long long factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;
    }
    return n * factorial(n - 1);
}

int main() {
    int num;
    
    printf("Enter a number: ");
    scanf("%d", &num);
    
    if (num < 0) {
        printf("Factorial is not defined for negative numbers.\n");
    } else {
        printf("Factorial of %d is %lld\n", num, factorial(num));
    }
    
    return 0;
}

Arrays: Storing Lists of Data

Definition: An array is a collection of elements of the same data type stored in contiguous memory locations. Arrays allow you to store multiple values under a single variable name and access them using an index number.
Key Concepts:
  • Element: Individual data item stored in an array
  • Index: Position number of an element in an array (starts from 0)
  • Size: Total number of elements an array can hold
  • Subscript: Another term for index, used with square brackets []

So far, we've used variables that can only hold one value at a time (e.g., int score = 95;). An array is a special variable that can hold multiple values of the same data type under a single name.

Think of it like a row of numbered boxes, where each box can hold one item.

2. Accessing Array Elements

You access elements in an array using their index. The index is the element's position, and it always starts at 0.

  • The first element is at index 0.
  • The second element is at index 1.
  • ...and so on.

Example 5: Accessing and printing elements

#include <stdio.h>

int main() {
    int scores[5] = {95, 88, 76, 92, 85};

    // Access the first element (at index 0)
    printf("The first score is: %d\n", scores[0]);

    // Access the third element (at index 2)
    printf("The third score is: %d\n", scores[2]);

    // You can also change an element's value
    scores[2] = 80;
    printf("The third score is now: %d\n", scores[2]);

    return 0;
}

Output:

The first score is: 95 The third score is: 76 The third score is now: 80

3. Using Loops with Arrays

Loops are perfect for working with arrays. You can use a for loop to go through each element of an array one by one.

Example 6: Printing all elements with a loop

#include <stdio.h>

int main() {
    int scores[5] = {95, 88, 76, 92, 85};
    int i;

    printf("All scores:\n");
    for (i = 0; i < 5; i++) {
        // The loop runs for i = 0, 1, 2, 3, 4
        printf("Score at index %d is %d\n", i, scores[i]);
    }

    return 0;
}

Output:

All scores: Score at index 0 is 95 Score at index 1 is 88 Score at index 2 is 76 Score at index 3 is 92 Score at index 4 is 85

Example 12: Reverse an Array

Write a program that reverses the elements of an array.

Solution:
#include <stdio.h>

int main() {
    int numbers[] = {1, 2, 3, 4, 5};
    int size = 5;
    
    printf("Original array: ");
    for (int i = 0; i < size; i++) {
        printf("%d ", numbers[i]);
    }
    printf("\n");
    
    // Reverse the array
    for (int i = 0; i < size/2; i++) {
        int temp = numbers[i];
        numbers[i] = numbers[size - 1 - i];
        numbers[size - 1 - i] = temp;
    }
    
    printf("Reversed array: ");
    for (int i = 0; i < size; i++) {
        printf("%d ", numbers[i]);
    }
    printf("\n");
    
    return 0;
}

Example 13: Count Frequency of Elements

Write a program to count how many times each element appears in an array.

Solution:
#include <stdio.h>

int main() {
    int numbers[] = {1, 2, 2, 3, 1, 4, 2, 1, 3};
    int size = 9;
    int visited[100] = {0}; // Assuming numbers are less than 100
    
    printf("Array: ");
    for (int i = 0; i < size; i++) {
        printf("%d ", numbers[i]);
    }
    printf("\n");
    
    printf("Frequency count:\n");
    for (int i = 0; i < size; i++) {
        if (visited[numbers[i]] == 0) {
            int count = 1;
            for (int j = i + 1; j < size; j++) {
                if (numbers[i] == numbers[j]) {
                    count++;
                }
            }
            printf("Number %d appears %d time(s)\n", numbers[i], count);
            visited[numbers[i]] = 1;
        }
    }
    
    return 0;
}

Strings: Working with Text

Definition: A string in C is a sequence of characters stored as an array, terminated by a special null character ('\0'). Strings represent text data and are fundamental for handling words, sentences, and textual information.
Key Terms:
  • String Literal: Text enclosed in double quotes ("Hello")
  • Null Terminator (\0): Special character that marks the end of a string
  • Character Array: Array of char data type used to store strings
  • String Length: Number of characters in a string (excluding null terminator)

In C, a string is simply an array of characters. There's one special rule: a string must end with a special character called the null terminator (\0). This character tells functions like printf where the string ends.

When you use double quotes (") to create a string, the compiler automatically adds this null terminator for you.

1. Declaring and Initializing Strings

This is just like creating a character array.

Example 14: Basic String Operations

Program Definition: This program demonstrates string creation and initialization in C. It shows two methods for creating strings: using string literals (double quotes) and character array initialization. This illustrates how strings are essentially character arrays with null terminators.
Key String Concepts Explained:
  • String Literal Method: "Hello" automatically includes the null terminator \0
  • Character Array Method: Manual initialization requires adding \0 explicitly
  • Null Terminator: The \0 character marks the end of the string for functions to recognize
  • String Storage: Each character occupies one byte in memory, stored consecutively
  • Format Specifier: %s in printf() displays the entire string until it encounters \0
#include <stdio.h>

int main() {
    // Method 1: Using double quotes (easy and common)
    // The compiler adds '\0' automatically.
    char greeting[] = "Hello";

    // Method 2: Initializing as a char array
    // You must add '\0' yourself!
    char farewell[] = {'B', 'y', 'e', '\0'};

    printf("Greeting: %s\n", greeting);
    printf("Farewell: %s\n", farewell);

    return 0;
}

Output:

Greeting: Hello Farewell: Bye

2. Common String Functions

To work with strings effectively, you need to use functions from the C standard library. To use them, you must include the <string.h> header file.

#include <string.h>

Here are a few of the most useful ones:

  • strlen(string): Returns the length of a string (not including the \0).
  • strcpy(destination, source): Copies the source string into the destination string.
  • strcat(destination, source): Concatenates (joins) the source string onto the end of the destination string.

Example 15: Using string functions

#include <stdio.h>
#include <string.h> // Don't forget this!

int main() {
    char firstName[50] = "John"; // Needs to be large enough for concatenation
    char lastName[] = "Doe";
    char fullName[100]; // Destination for copying

    // Get length
    printf("Length of firstName: %zu\n", strlen(firstName));

    // Copying a string
    strcpy(fullName, firstName); // fullName is now "John"
    printf("Full name after copy: %s\n", fullName);

    // Concatenating strings
    strcat(fullName, " "); // Add a space
    strcat(fullName, lastName); // Add the last name
    printf("Full name after concatenation: %s\n", fullName);

    return 0;
}

Output:

Length of firstName: 4 Full name after copy: John Full name after concatenation: John Doe

Example 16: Simple Username Greeting

Write a program that does the following:

  • Creates a char array to store a user's name (e.g., char name[50];).
  • Asks the user to enter their first name.
  • Reads the name using scanf.
  • Creates another char array for a greeting message (e.g., char message[100];).
  • Uses strcpy and strcat to build a greeting message like "Hello, [name]! Welcome!".
  • Prints the final greeting message.

Solution to Example 16:

#include <stdio.h>
#include <string.h>

int main() {
    char name[50];
    char message[100];

    // Ask for and read the user's name
    printf("Please enter your first name: ");
    scanf("%s", name);

    // Build the greeting message
    strcpy(message, "Hello, ");
    strcat(message, name);
    strcat(message, "! Welcome!");

    // Print the final message
    printf("%s\n", message);

    return 0;
}

Sample Output:

Please enter your first name: Alice Hello, Alice! Welcome!

Write a program that calculates the length of a string without using the strlen() function.

int main() {
    char str[100];
    int length = 0;

Example 18: Reverse a String

Write a program that reverses a string without using library functions.

Solution:
#include <stdio.h>
#include <string.h>

int main() {
    char str[100];
    char reversed[100];
    int length, j = 0;
    
    printf("Enter a string: ");
    scanf("%s", str);
    
    length = strlen(str);
    
    // Reverse the string
    for (int i = length - 1; i >= 0; i--) {
        reversed[j] = str[i];
        j++;
    }
    reversed[j] = '\0'; // Add null terminator
    
    printf("Original string: %s\n", str);
    printf("Reversed string: %s\n", reversed);
    
    return 0;
}

Example 19: Count Vowels and Consonants

Write a program that counts the number of vowels and consonants in a string.

Solution:
#include <stdio.h>

int main() {
    char str[100];
    int vowels = 0, consonants = 0;
    
    printf("Enter a string: ");
    scanf("%s", str);
    
    for (int i = 0; str[i] != '\0'; i++) {
        char ch = str[i];
        
        // Convert to lowercase manually
        if (ch >= 'A' && ch <= 'Z') {
            ch = ch + 32;  // Convert to lowercase
        }
        
        if (ch >= 'a' && ch <= 'z') {
            if (ch == 'a' || ch == 'e' || ch == 'i' || ch == 'o' || ch == 'u') {
                vowels++;
            } else {
                consonants++;
            }
        }
    }
    
    printf("String: %s\n", str);
    printf("Vowels: %d\n", vowels);
    printf("Consonants: %d\n", consonants);
    
    return 0;
}

Example 20: Check Palindrome

Write a program to check if a string is a palindrome (reads the same forwards and backwards).

Solution:
#include <stdio.h>
#include <string.h>

int main() {
    char str[100];
    int length, isPalindrome = 1;
    
    printf("Enter a string: ");
    scanf("%s", str);
    
    length = strlen(str);
    
    // Check if palindrome
    for (int i = 0; i < length/2; i++) {
        if (str[i] != str[length - 1 - i]) {
            isPalindrome = 0;
            break;
        }
    }
    
    if (isPalindrome) {
        printf("'%s' is a palindrome.\n", str);
    } else {
        printf("'%s' is not a palindrome.\n", str);
    }
    
    return 0;
}

Challenge: Simple Calculator

Write a program that asks the user for two numbers and an operation, then shows the result.

Solution:
#include <stdio.h>

int main() {
    float num1, num2, result;
    char op;
    
    printf("Enter first number: ");
    scanf("%f", &num1);
    
    printf("Enter operation (+, -, *, /): ");
    scanf(" %c", &op);
    
    printf("Enter second number: ");
    scanf("%f", &num2);
    
    if (op == '+') {
        result = num1 + num2;
        printf("%.2f + %.2f = %.2f\n", num1, num2, result);
    } else if (op == '-') {
        result = num1 - num2;
        printf("%.2f - %.2f = %.2f\n", num1, num2, result);
    } else if (op == '*') {
        result = num1 * num2;
        printf("%.2f * %.2f = %.2f\n", num1, num2, result);
    } else if (op == '/') {
        if (num2 != 0) {
            result = num1 / num2;
            printf("%.2f / %.2f = %.2f\n", num1, num2, result);
        } else {
            printf("Error: Cannot divide by zero!\n");
        }
    } else {
        printf("Invalid operation!\n");
    }
    
    return 0;
}
